# 小猴子找水果游戏说明文档
## 1. 游戏基本信息
- 游戏名称 ：小猴子找水果
- 游戏类型 ：双人休闲小游戏
- 游戏目标 ：通过10轮比赛，找到价值最高的水果，获得最多金币
- 获胜条件 ：10轮后，金币总数最高的玩家获胜
## 2. 游戏设置
### 2.1 背景与视觉效果
- 背景图片 ：使用 forest.jpg 作为游戏背景
- 标志类型 ：使用彩色气球作为水果标志
  - 气球颜色：红色（🔴）、蓝色（🔵）、紫色（🟣）
  - 动画效果：气球具有上下浮动的动画效果
  - 交互反馈：鼠标悬停时气球会放大并停止浮动
### 2.2 玩家设置
- 玩家数量 ：2名玩家
- 头像选择 ：玩家可以从4个猴子头像中选择
  - 头像文件： monkey1.jpg 、 monkey2.jpg 、 monkey3.jpg 、 monkey4.jpg
  - 选择流程：游戏开始前，玩家1和玩家2依次选择头像
### 2.3 游戏规则与流程
- 轮次设置 ：共10轮比赛
- 每轮流程 ：
  1. 屏幕上随机生成6-10个彩色气球
  2. 玩家轮流点击气球，每个气球会显示一个水果
  3. 每个玩家在每轮中可以选择2个水果
  4. 水果价值转换为金币，累加到玩家分数中
  5. 一轮结束后，进入下一轮，直到10轮全部完成
- 水果设置 ：
  - 水果等级：10个等级
  - 价值范围：1000金币 - 10000金币
  - 水果种类：苹果（🍎）、橙子（🍊）、香蕉（🍌）、葡萄（🍇）、草莓（🍓）、西瓜（🍉）、菠萝（🍍）、芒果（🥭）、椰子（🥥）、猕猴桃（🥝）
### 2.4 难度设置
- 难度选择 ：玩家可以选择每轮生成的气球数量
  - 选项：6个、7个、8个、9个、10个
  - 影响：气球数量越多，找到高价值水果的难度越大
  - 切换方式：点击难度选择按钮切换，当前选中的难度会高亮显示
### 2.5 游戏界面元素
- 玩家信息区 ：显示两名玩家的头像、名称和当前金币数
- 游戏信息区 ：显示游戏名称和轮次信息
- 难度选择区 ：提供难度选择按钮
- 游戏状态提示 ：显示当前轮到哪个玩家，格式为 (当前轮次/总轮次) 玩家名称，轮到你找水果啦！
- 结果模态框 ：游戏结束时显示获胜者信息和最终分数
- 头像选择模态框 ：游戏开始前用于选择头像
### 2.6 音效设置
- 胜利音效 ：游戏结束时播放胜利音乐
- 音效实现 ：使用Web Audio API生成简单的胜利音效
  - 音符序列：A4（440Hz）→ C5（523Hz）→ E5（659Hz）→ A5（880Hz）
  - 音量：0.3（适中）
  - 持续时间：1.2秒
- 浏览器兼容性 ：处理了浏览器自动播放政策，确保音效能够正常播放
## 3. 游戏控制
### 3.1 鼠标操作
- 选择头像 ：点击头像选项选择喜欢的猴子头像
- 确认选择 ：点击"确认选择"按钮确认头像选择
- 选择难度 ：点击难度按钮选择每轮气球数量
- 寻找水果 ：点击气球查找隐藏的水果
- 重新开始 ：游戏结束后，点击"再来一局"重新开始游戏
## 4. 技术实现
### 4.1 前端技术栈
- HTML5 ：游戏结构和界面元素
- CSS3 ：样式设计和动画效果
- JavaScript (ES6+) ：游戏逻辑和交互
- Web Audio API ：音效生成和播放
### 4.2 核心功能实现
- 随机位置生成 ：确保气球在游戏区域内随机分布
- 游戏状态管理 ：管理玩家分数、当前轮次、选中水果等状态
- 响应式设计 ：适配不同屏幕尺寸
- 模态对话框 ：实现头像选择和结果展示
- 动画效果 ：气球浮动动画和水果出现动画
### 4.3 浏览器兼容性
- 支持所有现代浏览器
- 处理了浏览器自动播放政策
- 适配移动设备和桌面设备

## 6. 游戏启动与访问
- 本地启动 ：
  1. 将游戏文件放在同一目录下
  2. 使用本地服务器启动，例如：
     ```
     python3 -m http.server 8000
     ```
  3. 在浏览器中访问 http://localhost:8000
- 在线访问 ：将游戏文件部署到服务器，通过URL访问
## 7. 游戏特色
1. 简单易上手 ：规则简单，适合所有年龄段玩家
2. 双人互动 ：支持两名玩家面对面游戏，增加互动乐趣
3. 视觉效果 ：精美的背景图片和流畅的动画效果
4. 音效反馈 ：游戏结束时的胜利音效增强游戏体验
5. 难度可调 ：玩家可以根据自己的水平调整游戏难度
6. 头像选择 ：个性化的头像选择增加游戏的趣味性
## 8. 注意事项
1. 确保所有图片文件（forest.jpg, monkey1.jpg-monkey4.jpg）都在同一目录下
2. 建议使用现代浏览器（Chrome, Firefox, Safari, Edge）进行游戏
3. 首次游戏时，需要点击页面任意位置激活音效
4. 游戏过程中，请勿刷新页面，否则游戏进度会丢失
