# 20260103

## 1. 游戏概述
游戏名称 ：小猴子找水果 游戏类型 ：双人对战找水果游戏 游戏平台 ：Web浏览器（支持桌面和移动设备） 开发技术 ：HTML5 + CSS3 + JavaScript

## 2. 游戏功能详情
### 2.1 核心功能 2.1.1 头像选择
- 游戏开始前，两位玩家依次选择自己的猴子头像
- 提供4种不同的猴子头像供选择
- 头像选择后会显示在游戏界面和胜利界面 2.1.2 难度选择
- 玩家可以选择游戏总轮数（6-10轮）
- 每轮固定出现10个待选气球
- 选择后直接开始游戏，无需重新选择头像 2.1.3 游戏界面
- 随机从forest1.jpg和forest2.jpg中选择背景图片
- 顶部显示游戏标题和玩家信息（头像、名称、分数）
- 中部显示当前轮次和轮到哪位玩家找水果
- 游戏区域随机分布10个彩色气球 2.1.4 气球和水果系统
- 每轮生成10个彩色气球（红、蓝、紫三种颜色）
- 气球采用悬浮动画效果，点击时有放大效果
- 每个气球内含一个随机水果，水果价值从1000到10000金币不等
- 点击气球后，气球消失，显示对应的水果和价值 2.1.5 游戏流程
- 两位玩家轮流点击气球寻找水果
- 每位玩家每轮可以选择2个气球
- 显示找到的水果和获得的金币
- 每轮结束后自动进入下一轮
- 达到最大轮数后，显示游戏结果 2.1.6 胜利判定
- 总分高的玩家获胜
- 显示获胜者头像和祝贺信息
- 播放胜利音乐（victory.mp3） 2.1.7 寻宝详情
- 显示每轮两位玩家找到的水果及其价值
- 标注每轮的获胜者（🏆）
- 显示最终总金币数
- 提供"再来一局"和"结束游戏"按钮
### 2.2 辅助功能 2.2.1 横屏提示
- 针对移动设备（手机、平板）显示横屏提示
- 确保游戏在最佳状态下运行
- 桌面设备自动隐藏提示 2.2.2 响应式设计
- 适配不同屏幕尺寸
- 气球位置会根据窗口大小自动调整
- 移动端支持垂直滚动，确保所有气球都能被选择 2.2.3 音频系统
- 胜利界面播放victory.mp3背景音乐
- 针对不同设备优化音频播放
- 支持多种音频播放备选方案，确保在各种浏览器中都能播放
## 3. 游戏规则设置
### 3.1 基本规则
- 游戏轮数 ：6-10轮（玩家可选择）
- 每轮气球数 ：10个
- 每位玩家每轮选择数 ：2个气球
- 先手规则 ：
  - 第一轮：随机选择先手玩家
  - 第二轮及以后：上轮分数低的玩家先选（分数相同则随机）
### 3.2 水果价值表
水果 表情 价值（金币） 等级 苹果 🍎 1000 1 橙子 🍊 2000 2 香蕉 🍌 3000 3 葡萄 🍇 4000 4 草莓 🍓 5000 5 西瓜 🍉 6000 6 菠萝 🍍 7000 7 芒果 🥭 8000 8 椰子 🥥 9000 9 猕猴桃 🥝 10000 10

### 3.3 胜利条件
- 游戏结束后，总金币数高的玩家获胜
- 若总金币数相同，则为平局
## 4. 操作指南
### 4.1 游戏开始
1. 玩家1选择头像，点击"确认选择"
2. 玩家2选择头像，点击"确认选择"
3. 选择游戏轮数（6-10轮）
4. 游戏自动开始
### 4.2 游戏过程
1. 根据提示，轮到的玩家点击屏幕上的气球
2. 每次点击会显示找到的水果和获得的金币
3. 每位玩家每轮可以点击2个气球
4. 两轮后进入下一轮，直到达到最大轮数
### 4.3 游戏结束
1. 显示获胜者信息和总分
2. 点击"寻宝详情"查看每轮记录
3. 在寻宝详情页面：
   - 点击"再来一局"开始新游戏
   - 点击"结束游戏"关闭所有窗口

# 20260102

## 1. 游戏基本信息
- 游戏名称 ：小猴子找水果
- 游戏类型 ：双人休闲小游戏
- 游戏目标 ：通过10轮比赛，找到价值最高的水果，获得最多金币
- 获胜条件 ：10轮后，金币总数最高的玩家获胜
## 2. 游戏设置
### 2.1 背景与视觉效果
- 背景图片 ：使用 forest.jpg 作为游戏背景
- 标志类型 ：使用彩色气球作为水果标志
  - 气球颜色：红色（🔴）、蓝色（🔵）、紫色（🟣）
  - 动画效果：气球具有上下浮动的动画效果
  - 交互反馈：鼠标悬停时气球会放大并停止浮动
### 2.2 玩家设置
- 玩家数量 ：2名玩家
- 头像选择 ：玩家可以从4个猴子头像中选择
  - 头像文件： monkey1.jpg 、 monkey2.jpg 、 monkey3.jpg 、 monkey4.jpg
  - 选择流程：游戏开始前，玩家1和玩家2依次选择头像
### 2.3 游戏规则与流程
- 轮次设置 ：共10轮比赛
- 每轮流程 ：
  1. 屏幕上随机生成6-10个彩色气球
  2. 玩家轮流点击气球，每个气球会显示一个水果
  3. 每个玩家在每轮中可以选择2个水果
  4. 水果价值转换为金币，累加到玩家分数中
  5. 一轮结束后，进入下一轮，直到10轮全部完成
- 水果设置 ：
  - 水果等级：10个等级
  - 价值范围：1000金币 - 10000金币
  - 水果种类：苹果（🍎）、橙子（🍊）、香蕉（🍌）、葡萄（🍇）、草莓（🍓）、西瓜（🍉）、菠萝（🍍）、芒果（🥭）、椰子（🥥）、猕猴桃（🥝）
### 2.4 难度设置
- 难度选择 ：玩家可以选择每轮生成的气球数量
  - 选项：6个、7个、8个、9个、10个
  - 影响：气球数量越多，找到高价值水果的难度越大
  - 切换方式：点击难度选择按钮切换，当前选中的难度会高亮显示
### 2.5 游戏界面元素
- 玩家信息区 ：显示两名玩家的头像、名称和当前金币数
- 游戏信息区 ：显示游戏名称和轮次信息
- 难度选择区 ：提供难度选择按钮
- 游戏状态提示 ：显示当前轮到哪个玩家，格式为 (当前轮次/总轮次) 玩家名称，轮到你找水果啦！
- 结果模态框 ：游戏结束时显示获胜者信息和最终分数
- 头像选择模态框 ：游戏开始前用于选择头像
### 2.6 音效设置
- 胜利音效 ：游戏结束时播放胜利音乐
- 音效实现 ：使用Web Audio API生成简单的胜利音效
  - 音符序列：A4（440Hz）→ C5（523Hz）→ E5（659Hz）→ A5（880Hz）
  - 音量：0.3（适中）
  - 持续时间：1.2秒
- 浏览器兼容性 ：处理了浏览器自动播放政策，确保音效能够正常播放
## 3. 游戏控制
### 3.1 鼠标操作
- 选择头像 ：点击头像选项选择喜欢的猴子头像
- 确认选择 ：点击"确认选择"按钮确认头像选择
- 选择难度 ：点击难度按钮选择每轮气球数量
- 寻找水果 ：点击气球查找隐藏的水果
- 重新开始 ：游戏结束后，点击"再来一局"重新开始游戏
