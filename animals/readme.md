已经成功为Animal游戏增加了以下功能：

1. 游戏打开弹窗 ：当游戏加载时，会显示一个欢迎弹窗，文字为"六六七七比一比，看看谁用的次数少"，用户需要点击按钮才能开始游戏。
2. 五级难度选择 ：
   
   - 1级：4×4网格（8对配对）
   - 2级：5×5网格（12对配对）
   - 3级：6×6网格（18对配对）
   - 4级：7×7网格（24对配对）
   - 5级：8×8网格（32对配对）
   - 每个难度级别都有不同的网格布局和卡片大小，确保良好的游戏体验
3. 连续玩三次提示 ：
   
   - 使用localStorage记录游戏次数
   - 每完成3次游戏后，会弹出提示窗口，文字为"六六七七谁的记忆力更好？"
   - 用户可以点击按钮继续游戏
其他改进：

- 扩展了动物表情库，支持更多配对
- 优化了响应式设计，确保在不同屏幕尺寸上都有良好表现
- 改进了游戏界面，增加了难度选择区域
- 修复了匹配数显示，根据当前难度动态更新总配对数
游戏现在已经具备了完整的功能，用户可以根据自己的能力选择合适的难度级别，享受记忆配对游戏的乐趣。
